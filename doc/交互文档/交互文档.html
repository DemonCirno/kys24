用户登录

  • ajax请求路径

    /kys24/userController/login

  • 请求方式

    POST

  • 请求参数 返回参数

    请求参数 参数类型 参数说明
    userPhone String 用户号码
    userPassword String 用户密码
返回参数 参数类型 参数说明
info String 状态码
  • 返回示例

用户不存在 登陆成功 密码错误
{“info”,0} {“info”,1} {“info”,-1}

用户注册

  • ajax请求路径

    /kys24/userController/register

  • 请求方式

    /POST

  • 请求参数 返回参数
请求参数 参数类型 参数说明
userId String 用户id
userName String 用户名
userPassword String 用户密码
userPhone String 用户号码
userAddress String 用户地址
type String 类型
  • 返回示例
用户已经注册 注册成功 注册失败
{“info”,-1} {“info”,1} {“info”,-1}

删除用户

  • ajax请求路径

    /kys24/userController/users/delete/{userId}

  • 请求方式

    /DELETE

  • 请求参数 返回参数

    请求参数 参数类型 参数说明
    userId Integer 用户id
    删除成功 删除失败
    {“upte”,1} {“upte”,-1}

短信

  • ajax请求路径

    /kys24/userController/message/{userPhone}

  • 请求方式

    /GET

  • 请求参数

请求参数 参数类型 参数说明
userPhone String 用户号码
发送失败 发送成功
{“rg”,0} {“rg”,newnum} Newnum为验证码
